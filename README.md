# Bezier surface

Это третья лабораторная работа курса **Компьютерная графика**. В ней был реализован алгоритм**поверхности Безье** по заданной сетке вершин, а так же алгоритмы поворота вокруг осей X и Y.  Отрисовка происходит в реальном времени без использования **OpenGL библиотек**(по заданию).
